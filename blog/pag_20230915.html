Catálogo con Flask

October 15, 2023 | Python Flask MySql SQLite jQuery CSS

sqlCatalogo es un catálogo de sentencias SQL, las cuales se pueden guardar y organizar además de ejecutarlas en una base de datos MySql para validarlas. Este es un ejercicio para practicar el uso de flask con bases de datos y por supuesto con Python, se usa HTML, CSS y jQuery para desplegar los datos. ( código en https://github.com/pmariaca/flask_sqlcatalog )

Con este catálogo se puede:

  • Salvar las sentencias SQL y ordenarlas por grupos. Las sentencias SQL se guardan en una base de datos sQlite. El menú tiene opciones para borrar uno o todos de los grupos donde se guardan éstas.
  • Se puede crear una conexión a base de datos MySql donde se listan todas las bases de datos a las que tenga acceso el usuario. Así pues, se puede seleccionar una base para validar la sentencia SQL
  • Las sentencias se escriben en un elemento HTML-textarea, y el resultado se despiega en una tabla, para lo que se usa el plugin https://www.datatables.net/ con el cual se puede generar un reporte CSV/PDF con el resultado.
  • Se pueden abrir varios tabs de búsqueda, para tener varias busquedas al mismo tiempo, así como otro grupo de tabs, para ver el resultado de las búsquedas, que a mi me ha resultado útil para comparar resultados.
