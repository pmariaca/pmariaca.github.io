November 4, 2022 | CSS javascript php xml

Debido a ue en mi trabajo requería tener a mano búsquedas recurrentes, decidí, como ejercicio personal hacer un cuaderno de notas con sql's para tener a mano la información y así repasar php, xml, js, css, mysql haciendo una pequeña aplicación llamada sqlPatoCatalog, fue hace ya varios años el código en https://github.com/axolote14/sqlPatoCatalog

sqlPatoCatalog es un catálogo donde se pueden guardar y organizar las búsquedas sql y ejecutarlas. Además se pueden generar reportes Csv o pdf

Con este catálogo se puede:

  • Al seleccionar una base de datos, se listan las tablas de lado derecho. Para esto use el jquery Selectr https://github.com/caseyWebb/selectr
  • Usar diferentes usuarios de mysql.
  • Salvar los sql's y ordenarlos por grupos. Estas búsquedas se guardan en un archivo XML. En el menú tiene opciones para borrar uno o todos de los grupos donde se guardan las sentencias sql.
  • Ver el resultado de la búsqueda, la cual es generada en una tabla, para lo que use el plugin datatables https://www.datatables.net/ con el cual se puede generar un reporte csv/pdf con el resultado.
  • Abrir varios tabs de búsqueda, para tener varias busquedas al mismo tiempo, así como otro grupo de tabs, para ver el resultado de las búsquedas, que a mi me ha resultado útil para comparar resultados.
  • Configurar tambén los el color con diferentes less de boostrap.
